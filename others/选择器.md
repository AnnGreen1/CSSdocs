# 选择器

## 简单选择器（Simple selector）

#### .class（Class selector）

选择所有 class="intro"的元素

```css
.intro {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```

```html
<div class="intro"></div>
```

#### #id（ID selector）

选择所有 id="firstname"的元素

```css
#firstname {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```

```html
<div id="firstname"></div>
```

#### \*（Universal selector）

选择所有元素

```css
* {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```

```html
<div class="intro">选择所有元素</div>
<div id="firstname">选择所有元素</div>
<div>选择所有元素</div>
```

tips:`*`也会选中 body 标签

#### element（Type selector）

选中所有 div 元素

```css
div {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```

```html
<div>div标签被选中</div>
<div>div标签被选中</div>
<p>p标签不会被选中</p>
```

## 组合器（Combinator）

#### element,element（Group selector）

选中所有 div 和 p 元素

```css
div,
p {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```

```html
<div>选择所有div元素和p元素</div>
<p>选择所有div元素和p元素</p>
```

#### element element（Descendant combinator）

选中 div 元素里的所有 p 元素

```css
div p {
  background-color: pink;
}
```

```html
<div>
  div标签
  <p>div标签里的p标签被选中</p>
</div>
<p>div外的p标签未选中</p>
```

#### element>element（Child combinator）

选择所有父级是 div 元素的 p 元素

```css
div > p {
  background-color: pink;
}
```

```html
<div>
  <p>父级是div被选中</p>
  <h1>
    <p>父级不是div未选中</p>
  </h1>
</div>
<p>父级不是div未选中</p>
```

#### element+element（Ajacent sibling combinator）

选择所有紧跟在 div 元素之后的第一个 p 元素

```css
div + p {
  background-color: pink;
}
```

```html
<div></div>
<p>第一个</p>
<p>第二个</p>
<p>第三个</p>
```

#### element~element（General cinling combinator）

选择类 todo 之后的每一个 ul 和选中类 done 之后的每一个 ul

```css
.wear ~ ul {
  color: pink;
}
.makeup ~ ul {
  color: hotpink;
}
```

```html
<div class="wear">穿搭</div>
<ul>
  <li>甜美可爱风</li>
  <li>复古英伦风</li>
  <li>文艺学院风</li>
</ul>
<div class="makeup">化妆</div>
<ul>
  <li>彩妆</li>
  <li>护肤</li>
  <li>香氛</li>
</ul>
```

tips：两种元素必须具有相同的父元素，但后者不必紧跟在前者的后面

## 属性选择器（Attribute Selector）

### 存在和值属性选择器（Presence and attribute selector）

#### [attribute]

选择所有带有 target 属性元素

```css
[target] {
  background-color: pink;
}
```

```html
<a href="https://github.com/AnnGreen1">https://github.com/AnnGreen1</a>
<a href="https://gitee.com/anngreens" target="_blank"
  >https://gitee.com/anngreens</a
>
<a href="https://www.aiit.edu.cn" target="_top">https://www.aiit.edu.cn</a>
```

#### [attribute=value]

选择所有使用 target="\_blank"的元素

```css
[target="_blank"] {
  background-color: pink;
}
```

```html
<a href="https://github.com/AnnGreen1">https://github.com/AnnGreen1</a>
<a href="https://gitee.com/anngreens" target="_blank"
  >https://gitee.com/anngreens</a
>
<a href="https://www.aiit.edu.cn" target="_top">https://www.aiit.edu.cn</a>
```

#### [attribute~=value]

选择类包含单词"hotpink"的所有元素

```css
.intro {
  width: 100px;
  height: 100px;
  background-color: pink;
}
[class~="hotpink"] {
  background-color: hotpink;
}
```

```html
<div class="intro hotpink"></div>
<div class="intro hotpink"></div>
<div class="intro"></div>
```

#### [attribute|=value]

```css
[class|="pink"] {
  width: 100px;
  height: 100px;
}
.pink {
  background-color: pink;
}
.pink-hotpink {
  background-color: hotpink;
}
.pink-deeppink {
  background-color: deeppink;
}
.pink-lightpink {
  background-color: lightpink;
}
```

```html
<div class="pink"></div>
<div class="pink-hotpink"></div>
<div class="pink-deeppink"></div>
<div class="pink-lightpink"></div>
```

### 子串值属性选择器（Substring value attribute selector）

#### [attribute^=value]

```css
[class^="pink"] {
  width: 100px;
  height: 100px;
}
.pink {
  background-color: pink;
}
.pinkhot {
  background-color: hotpink;
}
.pinkdeep {
  background-color: deeppink;
}
.pinklight {
  background-color: lightpink;
}
```

```html
<div class="pink"></div>
<div class="pinkhot"></div>
<div class="pinkdeep"></div>
<div class="pinklight"></div>
```

#### [attribute$=value]

选择类名以 pink 结尾的所有元素

```css
[class$="pink"] {
  width: 100px;
  height: 100px;
}
.pink {
  background-color: pink;
}
.hot-pink {
  background-color: hotpink;
}
.deep-pink {
  background-color: deeppink;
}
.light-pink {
  background-color: lightpink;
}
```

```html
<div class="pink"></div>
<div class="hot-pink"></div>
<div class="deep-pink"></div>
<div class="light-pink"></div>
```

#### [attribute*=value]

选择类名中包含 pink 的所有元素

```css
[class*="pink"] {
  width: 100px;
  height: 100px;
}
.pink {
  background-color: pink;
}
.hot-pink {
  background-color: hotpink;
}
.pink-deep {
  background-color: deeppink;
}
.light-pink-light {
  background-color: lightpink;
}
```

```html
<div class="pink"></div>
<div class="hot-pink"></div>
<div class="pink-deep"></div>
<div class="light-pink-light"></div>
```
